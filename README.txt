Bonjour,
Pour compiler le programme, 
Se placer dans le fichier “src” puis de lancer les commandes suivantes

COMPILER LE PROGRAMME
javac Main.java
LANCER LE PROGRAMME
java Main 

Les fichiers graph-011.paths et graph-001.costs contiendront les matrices, en format amatrix, 
demandées, une fois seulement que vous mettrez fin au programme, c’est à dire que presserez la touche zéro ‘0’

1er Partie du Programme :

--------------MOT_CLE-----------------------
DATA_ARC_POID_DPT_FR : Donne la matrice des couts entre chaque département. 
LISTE_ADJACENCE : Donne la liste d’adjacence, les liens qu’ils existent entre chaque vertex. 
MATRICE_ADJACENCE : Donne la matrice en bit des liens qu’ils existent entre chaque vertex. 
NON :  non en majuscule , si vous souhaitez passer directement à l’algorithme.
-------------------------------------
2eme Partie du Programme : 

---------------MOT-CLE----------------------
 Entrer un numero département de départ. 
 Entrer un numero département d’arriver. 
 ----------------------------------
 Le programme vous affiche le chemin le plus courts qu’il existe entre les deux, 
 puis affiche également son cout. 
 
 3eme Partie du Programme :
 ---------------MOT-CLE----------------------
 NON :  non en majuscule -> Reviens à la 2eme Partie du Programme.
 0   : zero -> Met fin au programme.
  ----------------------------------
  Bonne journée.
